# Project Context

## Founder Profile

Indian college student.
Needs fast personal finance management.
Variable monthly spending.
Wants premium experience.

## Main Pain Points

- random small expenses
- unclear spending trends
- money disappearing unnoticed
- poor purchase timing
- no enjoyable budgeting tools

## Product Mission

Create a finance assistant that feels intelligent, fast, private, and premium.

## Non-Negotiables

- offline-first
- mobile-first
- minimal friction
- beautiful UI
- actually useful

## Usage Frequency

Daily quick interactions.
Weekly deeper reviews.
Monthly planning.
