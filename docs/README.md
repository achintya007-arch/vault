# Vault

Vault is a premium offline-first personal finance operating system for personal use.

Built to outperform generic budgeting apps by focusing on:
- speed
- clarity
- privacy
- smart decision support
- beautiful mobile-first UX

## Core Goals

- Track spending effortlessly
- Understand money flow
- Make smarter buying decisions
- Build savings goals
- Feel in control financially

## Platform

Primary target:
- iPhone via Progressive Web App (PWA)

## Stack

- Frontend: Next.js
- Backend: FastAPI
- Database: SQLite
- Styling: Tailwind CSS

## Status

In active development.
