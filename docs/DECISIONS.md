# Architecture Decisions

## Platform

Using PWA first instead of native iOS.

Reason:
No Mac required.
Fast iteration.
Personal use friendly.

## Database

SQLite first.

Reason:
Offline-first.
Simple.
Reliable.

## Backend

FastAPI.

Reason:
Python familiarity.
Fast development.
Clean APIs.

## Frontend

Next.js.

Reason:
Strong ecosystem.
PWA support.
Good developer velocity.
