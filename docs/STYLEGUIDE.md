# Design Styleguide

## Visual Direction

Premium.
Minimal.
Modern.
Confident.

## Inspirations

- Apple
- Linear
- Notion
- Stripe dashboard polish

## UI Rules

- clean spacing
- no clutter
- one-thumb mobile use
- dark mode first
- subtle animations
- clear typography

## UX Rules

- common actions in 3 taps or less
- instant feedback
- zero confusion
- delightful microinteractions

## Tone

Smart.
Calm.
Professional.
Helpful.
